# Política de Seguridad

## Versiones Soportadas

| Versión | Soportada |
|---------|----------|
| latest  | ✅       |
| < 1.0   | ❌       |

## Reportar una Vulnerabilidad

**No abras un issue público para reportar vulnerabilidades de seguridad.**

Envía un email a: security@luisfelipevega.dev

Incluye:
- Descripción del problema
- Pasos para reproducir
- Impacto potencial
- Versión afectada

Responderemos en un plazo máximo de **72 horas**.

## Proceso

1. Confirmación de recepción en 72h
2. Evaluación y asignación de severidad
3. Fix desarrollado en rama privada
4. Release con patch + disclosure público coordinated

## Herramientas de Seguridad Activas

- GitHub Advanced Security (GHAS)
- CodeQL Analysis (JS/TS)
- Dependabot alerts
- Copilot Autofix
